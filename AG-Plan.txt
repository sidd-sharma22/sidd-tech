## Proposed Architecture & Stack

### Frontend (Client)
- **Framework:** React (initialized via Vite for speed).
- **Routing:** React Router DOM (Home, Projects, About, Resume, Contact).
- **Animations:** Framer Motion (for page transitions, scroll-reveals, and micro-interactions).
- **Styling:** Vanilla CSS (or Tailwind if requested) focusing on light-warm palettes and transparent glassmorphism effects (e.g., `backdrop-filter: blur()`).
- **Deployment Build:** Static output optimized for Vercel or Netlify.

### Backend (Server)
- **Framework:** Node.js with Express.js.
- **Email Service:** Nodemailer to handle POST requests from the Contact page and dispatch emails.
- **Database:** MongoDB (using Mongoose) to optionally log contact requests and potentially store project data (if a CMS approach is desired over hardcoded data).

## Proposed UI/UX Implementation Details

1. **Theme & Aesthetics:**
   - Backgrounds: Soft, warm gradients (e.g., soft peach, light beige, warm white).
   - Components: Glassmorphism panels with semi-transparent white backgrounds, rounded corners (e.g., `border-radius: 16px` to `24px`), and soft drop shadows.
   - Typography: Clean, modern sans-serif fonts (e.g., Inter, Outfit).
2. **Key Pages:**
   - **Home:** One-page scrolling feel with Hero, Skills, 3 Featured Projects, GitHub stats, and Contact CTA.
   - **Projects:** Grid layout with filtering and detailed case-study slugs (`/projects/:slug`).
   - **About:** Bio, Education, Timeline, and Skills Matrix.
   - **Resume:** HTML preview of the resume and a PDF download link.
   - **Contact:** Glassmorphic form connected to the Express backend.
